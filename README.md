## Installation

Clone Project

```bash
  git clone https://github.com/FauzanAldi/sharing-photo-app.git
```

Install project with composer

```bash
  composer install
```

## Configuration

add this to .env file

```javascript
STORAGE_URL=

JWT_SECRET=
JWT_ALGO=
```

## Running

To run this project

```bash
  php artisan serve
```
